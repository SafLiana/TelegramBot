# Travel Assistant Telegram Bot (TravelAi)

**Telegram-бот для путешественников**. Бот помогает планировать поездки, отслеживать цены на авиабилеты и узнавать информацию о достопримечательностях с помощью искусственного интеллекта.

### Основные возможности:
- ✈️ **Поиск ближайших рейсов** по заданному направлению с отображением актуальных цен.
- 🤖 **AI-гид по городам**: пользователь вводит название города, нейросеть (Giga-Chat) генерирует структурированную информацию о главных достопримечательностях, культуре и интересных местах.
- 🗺 **Просмотр популярных локаций**.

Бот обрабатывает текстовые запросы, взаимодействует с внешними API для поиска рейсов и использует большую языковую модель для генерации туристических справок.


**Основные пользовательские сценарии:**
1.  Пользователь вводит MOW LED 2026-03-14 и получает список ближайших рейсов с ценами.
2.  Пользователь вводит Город и ИИ присылает краткий путеводитель: Эйфелева башня, Лувр, Монмартр и т.д.
3.  Пользователь просматривает меню помощи и доступных команд.

## Характеристики

### 1. Поиск авиабилетов (Aviation API)
Принимает от пользователя пункт отправления и назначения.
*   Использует открытое API (*Aviasales*) для поиска актуальных данных.
*   Возвращает информацию:
    *   Номер и время рейса,
    *   Цена
    *   Авиакомпания.
    *   Количество пересадок

### 2. AI-гид по достопримечательностям
Пользователь отправляет название города, бот отправляет запрос в AI-модель.
*   Формируется промпт: *"Перечисли 10 главных достопримечательностей города и коротко опиши каждую"*.
*   Ответ от нейросети парсится и красиво оформляется в сообщении.


## Используемые API и сервисы

### 1. Aviasales API (Публичный)
Для поиска рейсов используется одно из публичных API

### 2. Giga Chat API (Провайдер ИИ)
Проект использует Giga Chat как готовый сервис для доступа к большим языковым моделям (LLM).
*   **Назначение:** Генерация контента о городах, создание "живого" диалога, обработка естественного языка.
*   **Плюсы:** Не требуется разворачивать тяжелые модели локально, высокая скорость инференса.
