# Travel Assistant Telegram Bot (TravelAi)

Telegram-бот для путешественников. Бот помогает планировать поездки, искать авиабилеты и получать информацию о городах с помощью искусственного интеллекта.

## Обзор проекта
Бот объединяет в себе несколько функций: поиск авиабилетов через внешнее API, получение структурированной информации о городах через AI-сервис GigaChat и анализ тональности сообщений пользователей с помощью дообученной модели-трансформера.

**Основные пользовательские сценарии:**
- Поиск авиабилетов по направлению и дате
- Получение краткого путеводителя по городу
- Анализ настроения сообщения при переводе

---

## Функциональность

### 1. Поиск авиабилетов
Бот принимает от пользователя пункт отправления и назначения в формате `MOW LED 2026-03-14`, использует Aviasales API для поиска данных и возвращает информацию о рейсах:
- Номер и время рейса
- Цена
- Авиакомпания
- Количество пересадок

### 2. AI-гид по городам
При получении названия города бот отправляет запрос к GigaChat API и возвращает структурированную информацию о главных достопримечательностях, культуре и интересных местах.

### 3. Анализ тональности сообщений
При переводе текста бот автоматически определяет эмоциональную окраску сообщения и добавляет соответствующий смайлик, а при негативном настроении предлагает помощь.

## Используемые сервисы и модели

### Внешние API

| API | Описание |
|-----|----------|
| **Aviasales API** | API для поиска авиабилетов. Позволяет получать информацию о рейсах: цены, время вылета, авиакомпании, количество пересадок и наличие билетов по заданному направлению и дате. |
| **GigaChat API** | Нейросетевая модель от Сбера для генерации текста. Используется для создания структурированных путеводителей по городам с описанием достопримечательностей, культуры и интересных мест. |

### Модели машинного обучения

| Модель | Описание |
|--------|----------|
| **cointegrated/rubert-tiny2** | Базовая модель-трансформер для русского языка. В проекте используется дообученная версия для решения задачи классификации текста — анализа тональности сообщений пользователей (определение позитивного, нейтрального или негативного настроения). |

## Технологический стек
- **Telegram-бот**: python-telegram-bot
- **Backend**: Python
- **API**: Aviasales, GigaChat
- **ML**: Transformers (Hugging Face), PyTorch
- **Язык**: Python
